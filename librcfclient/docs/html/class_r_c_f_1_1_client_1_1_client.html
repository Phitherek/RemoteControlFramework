Executes a command on the server.

Parameters
queryA command query - includes command name preceded by the hierarchy of groups that command belongs to, separated by : sign. Example: administration:network:mycommand, where mycommand belongs to the network group, and network group belongs to administration group which is top-level on the server.
stdout_targetA target string where command stdout should be placed.
stderr_targetA target string where command stderr should be placed.
paramProviderA function that provides additional parameters for the command (in response to PARAM protocol message). Takes no arguments, returns std::string.
Returns
Command exit code.
