Detailed Description

A class that can be used to create RCF server and contains useful RCF protocol methods.

Parameters
serviceA reference to boost::asio::io_service, that has to be constructed before server construction and is used for boost::asio communication (refer to boost::asio documentation for more information).
portA port the server is to be started at.
mainfnA pointer to the function, that returns void and takes two parameters - a pointer to RCF::Server::Server and a pointer to boost::asio::ssl::stream<boost::asio::ip::tcp::socket>. Inside the function define the server functionality using provided with this class RCF protocol methods.
